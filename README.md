#  SlotSwapper ‚Äì Smart Time Slot Management & Swapping Platform

SlotSwapper is a full-stack web application designed to simplify event management and slot swapping between users.  
It enables users to **create, manage, and exchange time slots** seamlessly through an intuitive interface.  
The system is secured with JWT authentication and powered by a Node.js + Express backend with MongoDB as the database.
